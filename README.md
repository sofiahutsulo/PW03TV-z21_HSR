# PW3
Android-застосунок мовою Kotlin для розрахунку прибутку сонячної електростанції з урахуванням точності прогнозування генерації.

## Опис
У межах практичної роботи №3 я реалізувала мобільний застосунок, який дозволяє оцінити вплив похибки прогнозу сонячної потужності на прибуток СЕС.

## Технології
- Kotlin  
- Android Studio
